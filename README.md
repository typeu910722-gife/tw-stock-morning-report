# 台股前一交易日晨報 V3

## 執行邏輯

程式會先向 TWSE 查詢最新已完成交易日，因此：

- 週六、週日執行：分析週五
- 週一 07:40 執行：分析上週五
- 週二 07:40 執行：分析週一
- 報告檔名使用實際交易日，不使用電腦執行日

## 使用

1. 解壓縮。
2. 雙擊 `run_report.bat`。
3. 報告位於 `outputs\YYYY\MM\台股日報_YYYYMMDD.html`。
4. 需要自動排程時，以系統管理員身分執行 `install_schedule_0740.bat`。

## 大盤資料

依序嘗試：

1. TWSE 新版日資料端點
2. TWSE 相容舊版端點
3. Yahoo Finance `^TWII` 備援

因此不會因單一端點欄位變動就整塊顯示 N/A。附錄會註明實際資料來源。
